# BOI-intrest-aggregator

CSV statements to filter out the intrest amount

## Table of Contents

1. [Introduction](#introduction)
2. [Installation](#installation)

## Introduction

It's a simple CLI tool.
Build specifically for Bank of India statements. The tool aggregate the intrests provided by bank.

## Installation

### Prerequisites

You need

- Node.js (version)
- npm or yarn

### Steps

1. Clone the repository:

   ```sh
   git clone https://github.com/maaz619/BOI-intrest-aggregator
   ```

2. Navigate to the project directory:

   ```sh
   cd your-repository
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

## Usage

The CLI takes two arguments

1. Script file
2. Statement folder

Make the script executable

```sh
chmod +x index.js
```

### To run

```sh
./index.js ./assets
```
